## init

</div>

<div className="py-3 flex flex-col gap-3">

Use the <CodeHighlight>init</CodeHighlight> command to create initial setup in your project.

This command installs required dependencies, configures CSS variables, and adds the <CodeHighlight>cn</CodeHighlight> util.

<CommandBlock
	className="mt-2"
	npmCommand="npx zed-ui init" 
	pnpmCommand="pnpm dlx zed-ui init" 
	yarnCommand="npx zed-ui init" 
	bunCommand="bunx --bun zed-ui init" 
/>

</div>

<div className="flex flex-col gap-5 mt-5">

### Options

<CodeBlock contentClassName="h-fit">

```md
Usage: zed-ui init

initialize your project and install dependencies.
```

</CodeBlock>

</div>

</div>

## add

</div>

<div className="py-3 flex flex-col gap-5">

Use the <CodeHighlight>add</CodeHighlight> command to add components and your dependencies in your app.

<CommandBlock
	npmCommand="npx zed-ui add" 
	pnpmCommand="pnpm dlx zed-ui add" 
	yarnCommand="npx zed-ui add" 
	bunCommand="bunx --bun zed-ui add" 
/>

</div>

<div className="flex flex-col gap-5 mt-5">

### Options

<CodeBlock contentClassName="h-fit">

```md
Usage: zed-ui add [components...]

select and add the components you need.

Arguments:
  components         enter the component name(s)
```

</CodeBlock>

</div>

</div>
