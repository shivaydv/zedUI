**Note:** We have the exact same installation process as [shadcn/ui](https://ui.shadcn.com/docs/installation/).

<Steps>

<Step title="Create project" number="1">
Run the `init` command to create a new Next.js project or to setup an existing one:

<CommandBlock
	npmCommand="npx shadcn@latest init"
/>

</Step>

<Step title="Add components" number="2">

You can now start adding components to your project.

<CommandBlock
	npmCommand="npx shadcn@latest add https://endorsy.in/registry/button.json"
/>


</Step>

<Step title="Import component" number="3">

The command above will add the `Button` component to your project. You can then import it like this:

<CodeBlock
fileName='Home.tsx'
contentClassName="h-fit"
simpleCode={`import { Button } from "@/components/ui/button"

export function Home() {
  return (
    <Button>
      Button
    </Button>
  )
}`}
>


```tsx
import { Button } from "@/components/ui/button"

export function Home() {
  return (
    <Button>
      Button
    </Button>
  )
}
```

</CodeBlock>

</Step>

</Steps>
